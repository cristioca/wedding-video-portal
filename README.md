# Wedding Video Client Portal (Next.js + Prisma + NextAuth)

## Setup
1. `npm install`
2. `.env` din `.env.example` cu `DATABASE_URL` È™i `NEXTAUTH_SECRET`
3. `npx prisma db push`
4. `npm run db:seed`
5. `npm run dev` la `http://localhost:3000`
